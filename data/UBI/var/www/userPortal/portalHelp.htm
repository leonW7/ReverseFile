Help Portal Information

VPN

SSL VPN Portal

Portal Info

This client portal provides remote access to the corporate network through the following features.

'SSL VPN Tunnel' facilitates in creating an encrypted tunnel to the corporate network.

'Port Forwarding' facilitates creating an encrypted tunnel for pre-defined applications on the corporate network.
 

Help VPN Tunnel

VPN

SSL VPN Portal

VPN Tunnel

An SSL VPN tunnel client provides a point-to-point connection between the remote client machine and the gateway of the corporate network. When a SSL VPN client is launched, a "network adapter" with an IP address, DNS and WINS settings is automatically created, which allows local applications to talk to services on the private network without any special network configuration on the remote SSLVPNClient machine.
This feature is presently supported on windows, Linux, MacOS X platforms.

Note:
* If you reload your browser, SSL VPN Tunnel Client will disconnect and then reconnect to the remote network.
* Installation/Uninstallation can be done only as "root".
* Java, Java Script, Active-X controls MUST be Enabled/Allowed in your browser settings.

Recommendations :
* Always follow the sequence : unInstall any active clients, logout and then close the browser. It is safe, easy to maintain the client, and can avoid errors during client installations, if recommendations are followed.
 

Help Port Forwarding

VPN

SSL VPN Portal

Port Forwarding


Port Forwarding is a light weight tunnel with port based encryption. Launching Port Forwarding allows to detect and re-route data sent from remote hosts to the corporate  SSL VPN gateway and hence to predefined applications running on private corporate networks. This feature is currently supported only on Windows platforms.

Note:
* The active connections can still persist even when browser is closed without uninstalling the port Forwarding client.
* Java, Java Script, Active-X controls MUST be Enabled/Allowed in your browser settings.

Recommendations :
* Always follow the sequence : unInstall any active clients, logout and then close the browser. It is safe, easy to maintain the client, and avoid errors during client installtions, if recommendations are followed.
